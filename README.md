# Voice Meeting Assistant

## Description
Voice Meeting Assistant is a web application designed to facilitate the management of voice meetings. It allows users to schedule, record, and manage meeting notes efficiently. The application provides a user-friendly interface for both administrators and regular users.

## Features
- Schedule meetings with date and time.
- Record audio of meetings.
- Store meeting notes and transcripts.
- User authentication for secure access.
- Admin panel for managing users and meetings.

## Installation
To set up the Voice Meeting Assistant locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/vedanth-raj/voice-meeting-assistant_v3.git
   cd voice-meeting-assistant_v3
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python main.py
   ```

4. Open your browser and navigate to `http://localhost:5000` to access the application.

## Usage
- **User Registration**: New users can register through the login page.
- **Scheduling Meetings**: Users can schedule meetings by providing the necessary details.
- **Recording**: Meetings can be recorded, and audio files will be saved in the designated directory.
- **Admin Panel**: Admins can manage users and view all scheduled meetings.
